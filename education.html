Relevant Courses

  1. STAT 331: Theory of Probability - This class gave me a good understanding of some of the basic concepts of probability that form the basis of statistics.
    1. Binominal distribution, Poisson distribution, geometric distribution, hypergeometric distribution, etc.
    2. Sampling theory
    3. Bayes' Theorem
    4. The Law of Large Numbers
  2. STAT 431: Theory of Statistics - This class introduced me to some methods of parameter estimation, model creation, and hypothesis testing.
    1. Confidence intervals
    2. Hypothesis Tests
    3. P-values
    4. Normal distribution, T distribution, standard normal distribution, χ2 distribution, F distribution
    5. Simple linear regression and correlation
  3. STAT 405: Introduction to Data Handling - This class showed me how data analysis is performed using statistical software, including the R and SAS programming languages.
    1. R programming language
    2. SAS programming language
    3. Use of programming to perform statistical analysis
    4. Importing data
    5. Data visualization in SAS and in R using the ggplot2 package
    6. Data manipulation/data cleaning
  4. STAT 435: Design and Analysis of Experiments - This class taught me how to create efficient experiments using a variety of designs.
    1. ANOVA with one factor, multiple factors, random effects, or interactions
    2. Latin squares
    3. Graeco-Latin squares
    4. Factorial designs
  5. STAT 449: Nonparametric Statistics - This class introduced me to nonparametric statistics, which are required for statistical analysis of certain problems, particularly problems with smaller sample sizes or for samples that deviate substantially from the normal distribution.
    1. Rank tests
    2. Permutation tests
    3. Wilcoxon Rank-Sum test
    4. Mann-Whitney Test
    5. Kruskal-Wallis test
    6. Signed rank test
    7. Fisher's Exact Test
  6. STAT 437: Regression and Time Series - This class taught me how to create, use, and evaluate linear regression models.
    1. Simple and multiple linear regression
    2. Model evaluation using R2, R2adj, and the PRESS statistic.
    3. Test of significance of parameters
    4. Comparison of full and reduced models
    5. Outlier detection and handling
    6. Model transformations
    7. Polynomial models
    8. Cross validation of models
